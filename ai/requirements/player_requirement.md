# Agregar Tidal Embed Player

Modificar el mockup (index.html) para agregar de la mejor manera el siguiente
código html que contiene un reproductor de una canción.

```
<iframe src="https://embed.tidal.com/tracks/100893567" width="500" height="120" allow="encrypted-media; fullscreen; clipboard-write https://embed.tidal.com; web-share" sandbox="allow-same-origin allow-scripts allow-forms allow-popups allow-popups-to-escape-sandbox" style="color-scheme: light dark" title="TIDAL Embed Player" />
```

En el código html se puede observar:

* Cover
* Nombre de la canción
* artista

Una vez dando click en el lado izquierdo a la canción en el panel derecho, a parte de 
la información, se debe poder reproducir la canción
